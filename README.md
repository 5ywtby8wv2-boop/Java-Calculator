 Java Calculator 

Description
A simple Java calculator that can add, subtract, multiply, and divide numbers.  
Built by OmarGaming2009z as a beginner project to showcase Java programming skills.

Features
- Addition, Subtraction, Multiplication, Division
- Handles invalid input
- Easy to use in terminal/console

How to Run
1. Download `Calculator.java`
2. Open in a Java IDE (Eclipse, IntelliJ, VS Code)
